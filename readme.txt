Database Backup Made Simple easily on single click

== Description ==

Simple Plugin to create a database backup. No bells and whistles, no bloat.

Are you worried every time WordPress puts a warning on your dashboard, "Please backup your database before proceeding?"
Just install this plugin, one click and it is done.

If you need to restore the database backup, use the phpMyadmin. Please, read our online documentation to more options and instructions.

The database backup creation is very fast and Osecure because this plugin use mySQLdump for create the backup and don't zip the file. 
This avoid the nightmare to restore it if the compressed zip (or tar) file is broken and you can't see nothing.
You can also, with the text editor, edit the backup file and take only one (or more) table to restore if necessary...

== Server Requirements ==
shell_exec php function available.
mySQLdump available.
phpMyAdmin to restore the Backup (usually the hosting panel has it.)
If you have questions, ask to your hosting company.

== Installation ==


1) Install via wordpress.org

2) Activate the plugin through the 'Plugins' menu in WordPress

or

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
